# 게임소모임 — 추가 기능 브레인스토밍

> 작성일: 2026-03-06

---

## 현재 구현된 기능

- 게임 갤러리 (태그 필터링, 최신/인기순 정렬)
- Google 로그인 (팀원 3명만 가능)
- 게임 업로드 (zip/html, 50MB 제한)
- 즐겨찾기/최근 플레이 기록
- 멤버별 카테고리 페이지
- 개인 포트폴리오 페이지 (역할, 소개, 기술스택, GitHub 링크)
- Command Palette (Ctrl+K)

| # | 기능 | 설명 | 난이도 | 백엔드 | 우선순위 |

| 1-5 | **게임 버전 관리** | 같은 게임 재업로드 시 v1, v2... 버전 이력 유지 | 어려움 | O | 나중에 |

## 우선순위 요약 — 추천 구현 순서

### Phase 1: 바로 하면 좋은 것 (쉬움 + 필수)

| 기능 | 예상 작업량 |
|------|-----------|
| 1-3 전체화면 모드 | ~30분 |
| 2-1 OG 메타태그 (SNS 미리보기) | ~30분 |
| 4-2 게임 공유 버튼 | ~1시간 |
| 4-3 팀 소개 페이지 | ~1시간 |
| 1-2 플레이 카운터 | ~1시간 |

### Phase 2: 핵심 가치 향상 (보통 + 필수/있으면 좋음)

| 기능 | 예상 작업량 |
|------|-----------|
| 1-1 게임 평점/별점 | ~2시간 |
| 2-5 게임 개발 후기 | ~2시간 |
| 1-4 게임 댓글 | ~3시간 |
| 2-3 기술 스택 뱃지 | ~1시간 |
| 3-5 공동 개발 표시 | ~2시간 |

### Phase 3: 나중에

- 게임 버전 관리
- 포트폴리오 PDF 내보내기
- 게임잼 이벤트 페이지
- 다국어 지원
- 포트폴리오 테마 선택
- 외부 프로젝트 링크
- 게임 임베드 코드
- 팀원 활동 로그
